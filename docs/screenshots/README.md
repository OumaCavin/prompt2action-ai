# Screenshot Guide

This directory contains screenshots for the Prompt2Action AI project README.

## How to Add Screenshots

### 1. Homepage Screenshot
1. Visit the live demo: https://prompt2action-ai-production.up.railway.app
2. Take a screenshot of the main homepage
3. Save as `homepage.png` in this directory
4. Uncomment the homepage screenshot line in the main README.md

### 2. Admin Panel Screenshot
1. Visit: https://prompt2action-ai-production.up.railway.app/admin
2. Login with `admin` / `admin`
3. Take a screenshot of the admin dashboard
4. Save as `admin-panel.png` in this directory
5. Uncomment the admin panel screenshot line in the main README.md

### 3. Workflow Interface Screenshot (Optional)
1. Create a sample workflow through the admin panel
2. Take a screenshot showing the workflow interface
3. Save as `workflow-interface.png` in this directory

## Screenshot Specifications
- **Format**: PNG or JPG
- **Size**: Maximum 1200px width for optimal display
- **Quality**: High resolution for crisp display on all devices
- **File Size**: Under 500KB for fast loading

## Usage in README
Once screenshots are added, uncomment the corresponding lines in the main README.md file:

```markdown
![Homepage Screenshot](docs/screenshots/homepage.png)
![Admin Panel Screenshot](docs/screenshots/admin-panel.png)
```
